    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1><a id="user-content-gatech-android-print" class="anchor" href="#gatech-android-print" aria-hidden="true"><span class="octicon octicon-link"></span></a>Gatech android print</h1>

<h1><a id="user-content-goal" class="anchor" href="#goal" aria-hidden="true"><span class="octicon octicon-link"></span></a>Goal</h1>

<p>The goal for this app is to help you using georgia tech's printing Service at ease
with you android phone.</p>

<h1><a id="user-content-notes" class="anchor" href="#notes" aria-hidden="true"><span class="octicon octicon-link"></span></a>notes</h1>

<p>This plugin unpacks minimal Debian installation with CUPS included,
and launches it using PRoot, then uses it for actual printing.</p>

<p>To generate Debian image, download repo <a href="https://github.com/pelya/cuntubuntu">https://github.com/pelya/cuntubuntu</a>
and launch script img-cups-jessie.sh from img directory.</p>

<p>You will need to do this from Debian/Ubuntu, and install few packages, specified in it's readme file.</p>

<p>PRoot can be downlaoded from <a href="http://proot.me/">http://proot.me/</a></p>

<p>Other than these things, the plugin contains only Java code, and no other magic.</p>

<p>There is no JNI or other weird stuff - Java code just calls lp, lpinfo, lpadmin etc commandline tools.</p>
